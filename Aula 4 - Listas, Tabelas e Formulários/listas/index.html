Glossário

Linguagem de Marcação de Hipertexto, usada para estruturar conteúdo na web.
CSS:
Folhas de Estilo em Cascata, usada para estilizar páginas web.
JavaScript:
Linguagem de programação que permite criar conteúdo interativo na web.
